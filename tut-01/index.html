Simple x-hr Component

Create a custom element that draws a horizontal line. Nothing fancy, just to demonstrate how a basic web component looks like.
  • Spec says element name MUST contain dash character somewhere.
  • HTML is NOT XML. DO NOT self-close a custom element tag as it will lead to unpredictable results. Close each tag with a closing tag.
